The session opened with a review of mini-project status and deadlines before outlining the day’s plan: a journey through embedding concepts into practical tasks, then an introduction to Langchain. The instructor recapped numerical text representations, vector space models, and semantics before detailing types of embeddings—word, sentence, image, and multimodal—and exploring real-world applications like retrieval-augmented generation (RAG), recommendation, clustering, and classification. Next, participants walked through Hugging Face assignments covering text generation, tokenization, prompt design, sentence similarity tests, translation pipelines, and word embedding APIs. In the second half, the focus shifted to Langchain: its role in connecting large language models to external tools, a demonstration of ChatOpenAI integration, and handling API keys. The class concluded with best practices for prompt length, token management, cost considerations, model selection, and a preview of building intelligent agents.
