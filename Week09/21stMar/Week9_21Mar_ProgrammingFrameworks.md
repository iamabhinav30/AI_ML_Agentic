This session guides participants through developing intelligent agents using LangChain. The instructor introduces the distinction between traditional functions/APIs and agentic AI, demonstrates setting up environment variables and obtaining API keys, and builds a weather API agent. Attendees integrate tools for live web search, configure an orchestrator pattern to select and invoke the right tool (calculator, search, sentiment, summarization), and explore open-source LLM hosting via Grok. Throughout, common setup and dependency issues are debugged, and practical best practices for prompt design and tool orchestration are reinforced.
