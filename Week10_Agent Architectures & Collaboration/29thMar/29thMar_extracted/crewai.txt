🧠 1. Finance Domain – Investment Risk Analyst Agent
📌 Problem Statement

Build an AI agent that takes a user query about a company or asset, performs market research, analyzes risks, and sends a structured investment summary.

Example User Query

“Analyze risks of investing in Tesla for the next 6 months and send me a summary.”

Expected Workflow
Generate search queries (news, financials, trends)
Fetch data (stock trends, macro factors, news sentiment)
Summarize insights
Identify risks + mitigation
Send Email / Slack alert
Sample Output
Risks: Market volatility, EV competition, regulatory pressure
Mitigation: Diversification, staggered investment
🛒 2. E-commerce Domain – Smart Purchase Advisor
📌 Problem Statement

Build an AI agent that helps users decide what to buy by comparing products, analyzing reviews, and triggering alerts.

Example User Query

“Should I buy iPhone 15 or wait for price drop? Notify me if price drops.”

Expected Workflow
Generate comparison queries
Fetch product data + reviews
Summarize pros/cons
Detect price trends
Trigger notification (email/Slack)
Output
Recommendation
Risk: Overpriced now
Action: Alert when price drops
